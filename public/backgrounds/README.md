# Background Images

This folder contains background images for the application.

## Supported formats:
- PNG
- JPG/JPEG
- WebP
- SVG

## Usage:
Place your background images in this folder and reference them in the code using:
`/backgrounds/your-image-name.jpg`

## Recommended dimensions:
- 1920x1080 or higher for full-screen backgrounds
- Optimize file sizes for web performance
